# Demo

Hello world!

This is the body. 
<button
  class="btn btn-primary"
  @click="showModal = true"
>
  Show modal
</button>

<modal
  title="Hello world!"
  v-model="showModal"
  @closed="showModal = false"
>
  This is the body.
</modal>
